## Project Overview

**Agri-Bot** is a highly knowledgeable and professional AI assistant specializing in all aspects of agriculture and farming. This bot is designed to provide detailed, accurate, and practical information on a wide range of topics, including **Crop Science**, **Pest and Disease Management**, **Soil Health**, **Irrigation**, and **Modern Farming Techniques**.

The core of the application is an **Edge Function** deployed on Supabase that leverages the **Gemini API** for generative AI capabilities.

## Tech Stack

This project is built using a modern, serverless stack:

* **Backend Deployment:** **Supabase Edge Functions** (using Deno)
* **AI Model:** **Gemini API** (`gemini-2.5-flash`)
* **Language:** **TypeScript**
* **Frontend:** **React Vite** (as specified in existing tech list, though the example client is plain HTML/JS)
